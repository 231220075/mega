# merge

**Merge changes**

### Usage

libra merge `<BRANCH>`

### Description

Incorporates changes from the named commits. The `<BRANCH>` could be a branch name (local or remote) or a commit hash.

**Currently, libra only supports fast-forward merge.**

### Options:

-   `-h`, `--help` Print help
